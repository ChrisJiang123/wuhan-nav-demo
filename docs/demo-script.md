# 演示脚本与已知限制

## 一、演示录屏脚本（约 4 分钟）

| 时长 | 场景 | 讲述重点 |
|---|---|---|
| 0:00–0:20 | 打开 App | 无广告、首页干净、直接进入搜索 |
| 0:20–0:50 | 搜索目的地 | 武汉城区 POI 搜索（走已验证地点） |
| 0:50–1:20 | 选择路线 | 展示 1–3 条路线的时间/距离/收费差异 |
| 1:20–2:00 | 进入导航 | 地图跟随、ETA、回中、静音、TTS 播报 |
| 2:00–2:30 | 路况着色 | 说明按道路等级着色（演示级，非实时） |
| 2:30–2:50 | 一键上报 | 2 步完成，突出克制的互助上报 |
| 2:50–3:20 | 设置与隐私 | 夜间模式、匿名上报、定位说明 |
| 3:20–3:40 | 运营后台 | 上报记录、审核状态 |
| 3:40–4:00 | 总结 | “开源自搭，一周可交付，后续可扩展” |

演示前准备：人工补录关键 POI；预设 2–3 条已验证起终点作为兜底，避免现场路由质量翻车。

## 二、已知限制（必须如实告知）

1. **仅 Android**，本周不支持 iOS。
2. **路径质量为演示级**：基于自建 OSRM 与 OSM 路网，可能出现绕路或非最优，不等同商业导航。
3. **无实时路况、无实时交通事件**：路况着色仅按道路静态等级；事件仅为后台人工录入的演示数据。
4. **POI 覆盖有限**：基于 OSM 数据，武汉本地部分地点可能缺失或过时。
5. **离线能力仅缓存级**：无离线全量地图包与离线路由。
6. **收费为估算/占位**。
7. **仅供小范围内部试用**，未做公开上架与对外服务所需的合规处理。
8. 地图数据 © OpenStreetMap contributors（ODbL）。
